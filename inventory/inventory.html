This page shows off the 5 prioritized User Stories as well as links to associated html wireframes

    User Stories

  1. User can access the "How to Play" page
  2. User can choose "Quick Play" instead of utilzing an account
  3. User can make "aliens"
  4. User can make "alien structures"
  5. User can set "tasks" for the turn
  6. User receives "reactions" relevant to the "tasks" they set at the end of turn

    Wireframe Guide

  1. Nav Bar for Item 1
  2. Main screen with "Quick Play" option for Item 2
  3. Gameplay screen with "alien" and "alien structure" build options for Items 3 and 4
  4. Gameplay screen also includes a "tasks" section for Item 5
  5. Reaction screen with "reactions" displayed
